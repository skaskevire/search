To build jar run gradle build command. Jar will be in build/libs folder.
external frameworks - spring boot (embedded spring mvc, jackson, jetty http server); bootstrap, jquery on user interface.


!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
!!! to run application you need to specify json file path !!!
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
example: 
java  -DfilePath=d:/data.json -jar gs-spring-boot-0.1.0.jar
